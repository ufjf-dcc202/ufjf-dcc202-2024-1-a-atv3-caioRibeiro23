Seções

Quando precisamos dividir o conteúdo em partes que agrupam um assunto ou uma funcionalidade da aplicação. Usamos seções.

Elementos de texto

O parágrafo é definido por <p> e indica um bloco de texto conciso. Notações tipográficas podem ser feitas com <i> e <b> para itálico e negrito. Mas, se o significado for ênfase ou força, use <em> e <strong>

Códigos

Se você precisar representar códigos ou conteúdo que o espaçamento importa, uso o pre. Por exemplo:

#include <iostream>

using namespace std;
int main (){
    cout << "Olá mundo!";
    return 0;
}
